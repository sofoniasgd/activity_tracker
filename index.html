About the Project

The inspiration for this project came from my own struggles with staying organized and on top of my daily activities. As someone who juggles multiple responsibilities, I found myself constantly searching for an efficient way to manage my tasks and monitor my progress. This personal experience drove me to create an Activity Tracker that could simplify task management and provide valuable insights into productivity.

This journey began several months ago and has been a significant part of my learning experience at ALX. The Activity Tracker project not only showcases my skills but also serves as a portfolio project for my software engineering studies. You can learn more about ALX here.
